# Librerías utilizadas en el proyecto

Cada dependencia declarada en `pubspec.yaml` se documenta a continuación como una ficha independiente para que puedas referenciarla fácilmente dentro del marco teórico.

- **flutter**  
  Framework base para construir interfaces nativas multiplataforma con un único código fuente.  
  Documentación: https://docs.flutter.dev

- **cupertino_icons**  
  Provee el set de íconos estilo iOS para widgets `Cupertino`.  
  Documentación: https://pub.dev/packages/cupertino_icons

- **google_fonts**  
  Carga dinámica de tipografías de Google Fonts sin empaquetar archivos locales.  
  Documentación: https://pub.dev/packages/google_fonts

- **flutter_bloc**  
  Implementa el patrón BLoC/Cubit para separar UI de la lógica de negocio y gestionar estados de forma reactiva.  
  Documentación: https://pub.dev/packages/flutter_bloc

- **equatable**  
  Permite comparaciones por valor en modelos/estados, evitando boilerplate en `==` y `hashCode`.  
  Documentación: https://pub.dev/packages/equatable

- **flutter_modular**  
  Sistema modular de ruteo e inyección de dependencias para estructurar features desacopladas.  
  Documentación: https://pub.dev/packages/flutter_modular

- **dio**  
  Cliente HTTP con interceptores, cancelación, reintentos y soporte JSON/API REST.  
  Documentación: https://pub.dev/packages/dio

- **image_picker**  
  Acceso a cámara y galería para seleccionar imágenes o videos desde el dispositivo.  
  Documentación: https://pub.dev/packages/image_picker

- **file_picker**  
  Selector de archivos genérico (PDF, documentos, etc.) con filtros por extensión.  
  Documentación: https://pub.dev/packages/file_picker

- **image**  
  Procesamiento de imágenes en Dart (redimensionar, recortar, filtros, manipulación de píxeles).  
  Documentación: https://pub.dev/packages/image

- **video_compress**  
  Compresión local de videos para reducir tamaño antes de subirlos a la red.  
  Documentación: https://pub.dev/packages/video_compress

- **shared_preferences**  
  Almacenamiento persistente clave-valor para configuraciones ligeras y datos de sesión.  
  Documentación: https://pub.dev/packages/shared_preferences

- **intl**  
  Herramientas de internacionalización: formatos de fecha, número, mensajes y localización.  
  Documentación: https://pub.dev/packages/intl
